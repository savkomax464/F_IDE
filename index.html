How do I implement a debounce function in JavaScript?
Here's a clean implementation of a debounce function:
function debounce(fn, delay = 300) { let timerId; return function(...args) { clearTimeout(timerId); timerId = setTimeout(() => { fn.apply(this, args); }, delay); }; } // Usage example const handleSearch = debounce((query) => { console.log(`Searching: ${query}`); }, 500);
Can you add TypeScript types to it?
Sure! Here's the TypeScript version with generics:
type AnyFunction = (...args: any[]) => void; function debounce<T extends AnyFunction>( fn: T, delay: number = 300 ): T { let timerId: ReturnType<typeof setTimeout>; return ((...args: Parameters<T>) => { clearTimeout(timerId); timerId = setTimeout(() => fn(...args), delay); }) as T; }
